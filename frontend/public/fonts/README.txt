BB Casual Pro Font Files
========================

This folder is where you place your BB Casual Pro font files.

Purchase the typeface at: https://blambot.com/products/bb-casual-pro

After purchase, export and place these two files here:
  - bbcasualpro-regular.woff2
  - bbcasualpro-medium.woff2

The @font-face declarations in src/index.css will pick them up automatically
on next build. No code changes needed.

Until the files are present, the site falls back to Inter.
